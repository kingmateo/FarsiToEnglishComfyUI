🛠️ راهنمای دریافت API Key از DeepSeek

آموزش کامل دریافت کلید API برای استفاده در نود ترجمه ComfyUI

1

ثبت‌نام در پلتفرم DeepSeek

ابتدا به وبسایت DeepSeek مراجعه کنید:

https://platform.deepseek.com

روی دکمه "Sign Up" کلیک کرده و حساب کاربری ایجاد کنید.

2

ورود به پنل کاربری

پس از ثبت‌نام و تأیید ایمیل، وارد پنل کاربری خود شوید.

وارد پنل که شدید برید به بخش مدیریت کلید ها

3

دریافت API Key

در منوی کناری، روی گزینه "API Keys" کلیک کنید:

سپس روی این بخش بزنید تا پنجره ساخت کلید باز بشود

حالا کلید را باید ببرید و جای گذاری کنید تا بتونید از هوش مصنوعی به صورت انلاین و مستقیم استفاده کنید

4

تنظیمات API Key

برای کلید API خود یک نام انتخاب کنید (مثلاً: ComfyUI-Translator)

دسترسی‌های لازم را انتخاب کنید (معمولاً همه دسترسی‌ها لازم است)

اگر پنجره دسترسی باز نشد هیچ مشکلی ندارد

5

کپی کردن API Key

پس از ایجاد، کلید API شما نمایش داده می‌شود. حتماً آن را کپی کنید:

sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

کلید رو کپی کنید

⚠️ هشدار امنیتی:

این کلید را با کسی به اشتراک نگذارید و در محیط امن ذخیره کنید.

6

استفاده در نود ComfyUI

کلید API را در فایل مربوطه جایگزین کنید:

# در فایل farsi_to_english_deepseek.py
DEEPSEEK_API_KEY = "کلید-API-شما-اینجا"

یا از طریق فایل .env استفاده کنید:

# ایجاد فایل .env در روت ComfyUI
DEEPSEEK_API_KEY=کلید-API-شما-اینجا

کد رو اصلاح کنید و تمام

📝 نکته مهم:

برای ویرایش کد های این پلاگین از نوتپد خود ویندوز هم میتونید استفاده کنید

📝 نکات مهم:

  • API Key شما مانند رمز عبور است - آن را امن نگه دارید
  • در صورت لو رفتن کلید، بلافاصله آن را غیرفعال کنید
  • محدودیت استفاده ماهانه را بررسی کنید
  • برای مشکلات فنی با پشتیبانی DeepSeek تماس بگیرید
